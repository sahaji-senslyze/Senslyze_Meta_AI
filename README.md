# **Senslyze_Meta_AI**

A powerful framework to build fully customizable chatbots that leverage the **Llama Model (Llama 3.1 70B)** for enhanced user interaction. Senslyze_Meta_AI is designed to understand user intent seamlessly and provide a tailored UI experience. Whether it's integrating third-party APIs or custom tooling, Senslyze_Meta_AI delivers an intuitive, agent-driven conversation flow.

## ğŸ› ï¸ **Getting Started**

Follow the steps below to set up and run Senslyze_Meta_AI:

1. **Clone the repository**:  
   ```bash
   git clone https://github.com/sahaji-senslyze/Senslyze_Meta_AI.git
   ```

2. **Install dependencies**:  
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Flask application**:  
   ```bash
   flask run
   ```

4. **Configure your environment**:  
   - Set up AWS for cloud deployment.
   - Connect Groq and Bedrock for AI model interactions.
   - Integrate with ElasticSearch for seamless search functionalities.
   - Set up your WhatsApp and Shopify API credentials.
